API Documentation

GET /api/maps

Fetches all map data from the index.json file.

Response: A JSON array of map metadata.

Example:
[
  {
    "MapUUID": "123e4567-e89b-12d3-a456-426614174000",
    "MapName": "Sample Map",
    "MapDescription": "A sample map for demonstration purposes.",
    "MapType": "PvP",
    "MapDeveloper": "DeveloperName",
    "DateCreated": "2024-08-19"
  }
]

GET /api/search?q=query

Searches for maps matching the query in MapName, MapDescription, MapDeveloper, or MapType.

Query Parameters:

  • q - The search query string.

Response: A JSON array of matching map metadata.

POST /api/upload

Uploads a map file in .zip format and extracts metadata from the MetaData.json file inside the .zip.

Form Data:

  • map - The .zip file to be uploaded.

Response: A JSON object with a success message.

Example:
{
  "message": "Map uploaded and metadata extracted successfully"
}

GET /api/download/:uuid

Downloads a map file by its UUID.

Path Parameters:

  • uuid - The UUID of the map to download.

Response: The map .zip file as a download.

If the map is not found, returns a 404 error with a JSON message.
